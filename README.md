
# RayTracer
Raytracer from scratch developed by my self using Javaa and Object Oriented Programing.
Materials:
  Lambert
  Reflective
  Refractive
  
Lights:
 - Directional Light
 - Point Light
 - Spot Light
  
Objects:
 - Camera 
 - Sphere
 - Polygons
 - Any .Obj file you require 
  

I implemented threads to accelerate the process.
